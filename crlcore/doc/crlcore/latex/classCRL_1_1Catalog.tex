\doxysubsection{Detailed Description}
A Registry to store Alliance Cell metadatas. 

The \mbox{\hyperlink{classCRL_1_1Catalog}{Catalog}} is built from the various CATAL files from the library directories and enriched with the current state of Cell in the database.
