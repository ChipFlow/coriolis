\doxysubsection{Detailed Description}
Holds all the Alliance environment variables. 

This object is not to be used alone, but only as an attribute of \mbox{\hyperlink{classCRL_1_1AllianceFramework}{Alliance\+Framework}}.

Although we constantly refer of the contents of this object as {\itshape environement variables}, they may (and should) be initialized through the Coriolis Python environement loader or legacy XML parsers.
